Gear Builder API

Simple frontend and backend RESTful project

In college I had the opportunity to learn about web services. As an assignmen I had to make a RESTful API backend using Java and frontend of my choice. Decided to make a simple gear builder website, based on MMORPG called Runescape.

Also implemened Firebase authorazation system, for logging in and building gear only on your unique ID, so you could only edit or delete your gear presets.

Whole backend was made with Java and Maven, it's a simple project without any sql databases, so all data was stores inside the program classes.

This project is pretty small, so only a few functions were added. User can add gear, by selecting wanted items. Edit gear that he added or delete it. Also users may add new items to the database, but no icons were implemented yet.

Technologies used:
IntelliJ IDEAJavaMavenGlassFish 5.0.0RESTful web servicesVisual Studio CodeHTMLCSSJavaScriptFirebasePostman
